## 📖 Overview

The **Smart Navigation System** is a next-generation web-based platform specifically engineered to prioritize driver safety. By intelligently analyzing multiple hazard factors—such as high-risk and accident-prone zones, real-time severe weather conditions, and proximity to emergency medical facilities—the system delivers routes that are secure, balanced, and fast.

Built with a stunning, modern, glassmorphism-inspired interface, it offers a deeply immersive and responsive user experience right in the browser, ensuring drivers have the best situational awareness on the road.

---

## ✨ Premium Features

- 🛡️ **Safety-First Routing Algorithm**
  Dynamically calculates the safest paths by identifying and dodging documented blockages and historical accident hotspots.
- 🌦️ **Live Weather & Environmental Monitoring**
  Integrates real-time weather constraints directly into routing decisions.
- 🏥 **Proximity Medical Assistance**
  Instantly plots nearby hospitals and provides quick-access routing in emergency scenarios.
- 🗣️ **Intelligent Voice Assistant**
  Hands-free operation through built-in voice commands and audible navigation alerts.
- 🚨 **Emergency SOS Protocol**
  A dedicated quick-access panel to simulate direct dialing to local authorities, ambulance services, and highway patrols.
- 🌗 **Vibrant Dark / Light Theming**
  A beautifully crafted, modern UI featuring blur backdrops, gradient buttons, micro-animations, and seamless theme switching.

## 🏗️ Project Architecture

To ensure maintainability, the project follows a clean separation of concerns:

| File | Role | Description |
| :--- | :--- | :--- |
| 📄 `index.html` | **Structure** | The semantic entry point that houses the Map DOM, Control Panels, and Modal overlays. |
| 🎨 `style.css` | **Presentation** | The comprehensive design system containing all CSS variables, animations, and responsive layouts. |
| ⚙️ `script.js` | **Logic** | The core JavaScript engine powering Leaflet map integrations, simulations, geocoding, risk zone detection, and user interaction. |
| 🐍 `apply_modern_theme.py` | **Utility** | An automation script previously used to compile and inject modernized UI components into legacy HTML files. |

## 🚀 Getting Started

Running the Smart Navigation System natively requires no complex build pipelines or server-side setups for the frontend simulation.

### Prerequisites
- Any modern web browser (Google Chrome, Mozilla Firefox, Apple Safari, Microsoft Edge).

### Installation
1. Clone or download the repository to your local machine:
   ```bash
   git clone https://github.com/sathishr-ai/Smart-Navigation-System-for-Accident-Prone-Detection.git
   ```
2. Navigate into the root directory of the project:
   ```bash
   cd "Smart Navigation System"
   ```
3. Open the `index.html` file using your preferred web browser:
   - On Windows: Double-click `index.html`
   - Alternatively, you can run a local server:
     ```bash
     python -m http.server 8000
     ```
     Then navigate to `http://localhost:8000`
