## ✨ Premium Features

- 🛡️ **Safety-First Routing Algorithm:** Dynamically calculates the safest paths by identifying and dodging documented blockages and historical high-risk accident hotspots.
- 🌦️ **Live Environmental Monitoring:** Integrates real-time weather constraints directly into routing decisions, warning drivers of severe storms or low visibility.
- 🏥 **Proximity Medical Assistance:** Instantly plots nearby hospitals and provides quick-access emergency routing in critical scenarios.
- 🗣️ **Intelligent Voice Assistant:** Hands-free operation through built-in voice commands and audible navigation alerts to keep your eyes on the road.
- 🚨 **Emergency SOS Protocol:** A dedicated quick-access panel to simulate direct dialing to local authorities, ambulance services, and highway patrols.
- 💎 **Enterprise UI/UX:** A stunning, modern, glassmorphism-inspired interface featuring blur backdrops, gradient buttons, micro-animations, and seamless dark/light theme switching.

## � Setting Up the Project

Because the project strictly uses client-side web technologies and native geographical mapping APIs, **no server or complex build pipelines are required.**

1. **Clone the repository**
   ```bash
   git clone https://github.com/sathishr-ai/Smart-Navigation-System-for-Accident-Prone-Detection.git
   ```
2. **Open the project**
   Navigate into the directory. You can simply double-click `index.html` to open it locally directly in your web browser (Chrome, Safari, Edge, Firefox).
3. *(Optional)* **Run a local server**
   If you want to view it via a local development server to mirror browser CORS handling, you can use Python:
   ```bash
   python -m http.server 8000
   ```
   Then navigate to `http://localhost:8000/index.html`


## � How To Use

1. **Launch the platform** by navigating to the live URL or double-clicking the local `index.html` file.
2. **Search Locations**: Enter your starting point and destination in the floating search panel. Use the swap button to quickly reverse routes.
3. **Select Routing Strategy**: Choose between "Safest Route", "Balanced Route", or "Fastest Route" toggles.
4. **Configure Overlays**: Toggle checkboxes to show or hide Risk Zones, Weather Constraints, Regional Hospitals, or Speed Camera Alerts directly on the map engine.
5. **Drive Mode**: Click "Find Safe Route" to preview the journey outline in the Route Summary modal. Click "Start" to enter the live dashboard showing simulated Speedometer, time limits, and active ETA tracking.
